# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release of VoltCheck
- Real-time battery current monitoring
- Voltage and temperature tracking
- Battery level and health display
- Min/Max current tracking
- Dark mode support
- Multi-language support (English, Indonesian)
- Customizable refresh intervals (0.5s, 1s, 2s, 5s)
- Background monitoring service
- Smart notification system
  - Battery full alerts
  - Low battery/slow charging warnings
  - Fast charging detection
  - High temperature alerts
- Custom battery level alarms
- Time to full charge estimation
- Design capacity configuration
- Unit selection (mA/Ampere)
- Decimal precision toggle
- Data export to CSV
- Historical data logging
- Session summary tracking
- Material Design UI
- Comprehensive settings page
- About developer information
- Terms of service
- Privacy policy

### Technical
- MVVM architecture implementation
- Service-based background monitoring
- SharedPreferences for settings persistence
- File-based data logging system
- LocaleHelper for multi-language support
- NotificationUtil for notification management
- MPAndroidChart integration for data visualization

### UI/UX
- Modern Material Design interface
- Card-based layout design
- Dark mode with custom color scheme
- Intuitive settings organization
- Professional icon set
- Smooth animations and transitions
